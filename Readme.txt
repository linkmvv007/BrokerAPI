1. Для доступа к курсам валют используется сервис fixer.io.
для получения курсов доллара на дату 2023-04-12 формируется урл вида:
https://api.apilayer.com/fixer/2023-04-12?base=USD&symbols=RUB,EUR,GBP,JPY
для доступа к сервису необходимо задать ключ доступа, описан в appsettings.json 
в разделе  "ApiLayer" ключ ApiKey

2. Кэширование реализовано при помощи Redis.
Необходим доступ к Redis серверу. 
Настройки подключения к серверу Redis описаны в файле appsettings.json в разделе  "Redis"

3. Используемые NuGet пакеты  в проекте:
- FluentValidation: для валидации параметров запроса API;
- Mediator: для слабой связности между классами;
- nUnit: для написания unit тестов;
- Mock - для тестов, чтобы не делать тяжелые вызовы внешнего API;
- Microsoft.Extensions.Caching.StackExchangeRedis: для доступа к серверу Redis;
